FAQ

  1. Why my alarm is not ringing?

    Answer: The battery should not be optmized for SmartWaker and allow Smart Waker to auto start, If you're running any task killer/management apps like Greenify, please either disable them or whitelist Smart Waker in their settings.

    SOLUTIONS depending on your DEVICE

    unmonitored app example:

    • SAMSUNG, Setting -> Device Maintenance -> Battery -> Unmonitored apps -> add Smart Waker
    • HUAWEI, Setting -> Advance Setting -> Battery Manager -> Protected apps -> Smart Waker -> Enable Switch

    allow auto start example:

    • HUAWEI, Setting -> Startup Manager -> Allow Smart Waker to run after startup
